Eulerian and Hamiltonian Paths

Eulerian and Hamiltonian paths are specific types of paths in a graph, significant in various applications.

Eulerian Paths and Circuits

An Eulerian path is a path that visits every edge exactly once. An Eulerian circuit is an Eulerian path that starts and ends at the same vertex.

Conditions for Existence

  • An undirected graph has an Eulerian circuit if and only if all vertices have even degrees.
  • An undirected graph has an Eulerian path (but not a circuit) if exactly two vertices have odd degrees.

Hamiltonian Paths and Cycles

A Hamiltonian path is a path that visits every vertex exactly once. A Hamiltonian cycle is a Hamiltonian path that starts and ends at the same vertex.

Determining whether such paths exist is generally NP-complete, making it computationally challenging.
