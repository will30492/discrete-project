Graph Coloring

Graph coloring involves assigning colors to vertices so that no two adjacent vertices share the same color. The minimum number of colors needed is called the chromatic number.

Applications

  • Scheduling: Assigning time slots or resources without conflicts.
  • Register Allocation: In compilers, assigning variables to CPU registers.

Coloring Algorithms

The problem of finding the chromatic number is NP-hard. Greedy algorithms can be used to find an approximate solution.

Greedy Coloring Algorithm

  1. Order the vertices (e.g., randomly or by degree).
  2. For each vertex:
    1. Assign the smallest color number not used by its neighbors.
